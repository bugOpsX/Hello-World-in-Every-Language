# hello.bat

A tiny Windows batch file that prints "Hello, World!" to the console.

## Requirements
- Windows with Command Prompt (cmd.exe)

## How to run
- Double-click the file in Windows Explorer, **or**
- Open Command Prompt in the folder and run:
hello.bat
