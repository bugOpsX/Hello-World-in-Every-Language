🌍 Hello World Collection — Contribution Guide

Add Hello World programs in new languages, learn syntax differences, and join a global collaborative project.

📁 Folder Structure

languages/
└── Language-Name/
    ├── hello.<ext>    
    └── README.md

🌟 Why Contribute?

  • Learn new programming languages hands-on
  • Understand syntax differences across languages
  • Showcase your coding portfolio
  • Be part of a global learning community

🛠 How to Contribute

  1. Fork the repository.
  2. Clone it and create a new branch: 
    git clone https://github.com/<your-username>/hello-world-collection.git
cd hello-world-collection
git checkout -b add-<language>
  3. Create a new folder inside languages/ named after your language.
  4. Add your files: 
    hello.<ext>     
README.md
  5. Commit & push changes: 
    git add .
git commit -m "Add Hello World for <Language>"
git push origin add-<language>
  6. Submit a Pull Request (PR) and wait for review 🚀
